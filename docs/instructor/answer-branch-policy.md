# Answer Branch Policy

`NN-answer`는 참고 정답과 강사용 비교 기준입니다.
학생 첫 화면에서 과도하게 노출하지 않습니다.

## 허용되는 위치

- 강사용 문서
- 운영자 체크리스트
- 시퀀스 리뷰 문서
- 토픽 레포의 하단 비교 안내

## 피해야 할 위치

- 학생용 README 첫 시작 단계
- 실습 시작 명령 바로 위
- Visual Lab의 첫 행동 유도 영역
- 긴 정답 코드가 포함된 중앙 문서

## 학생에게 안내하는 표현

권장:

```text
막혔거나 실습을 마친 뒤 NN-answer와 비교합니다.
```

피해야 함:

```text
먼저 NN-answer를 열어 정답을 확인합니다.
```

## 관리 기준

- answer 브랜치에는 완성 코드와 비교 문서가 있어야 합니다.
- 학생용 starter와 answer의 차이는 수업 목표와 직접 연결되어야 합니다.
- 정답 코드 전체를 중앙 레포 문서에 복사하지 않습니다.
